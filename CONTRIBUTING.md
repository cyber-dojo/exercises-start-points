# Contributing to cyber-dojo's Exercises

The format is very simple. Each sub directory of `start_points/`  (eg Fizz_Buzz) has two files:
- `readme.txt` containing the instructions for exercise
- `manifest.json` naming the readme file and specifying the exercise display name.
  For example:
  ```json
  {
    "display_name": "Fizz Buzz",
    "visible_filenames": [ "readme.txt" ]
  }
  ```

Please clone and submit your pull-request.

Please email [support@cyber-dojo.org](mailto:support@cyber-dojo.org 'Other ways you can contribute to the LTFs') if you need help.
