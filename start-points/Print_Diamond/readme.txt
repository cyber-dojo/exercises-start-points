Given a letter print a diamond starting with 'A' with the supplied letter at the widest point.
For example, this is the diamond for 'E'.

    A
   B B
  C   C
 D     D
E       E
 D     D
  C   C
   B B
    A

The example for 'A' is in the file example/a
The example for 'C' is in the file example/c
The example for 'E' is in the file example/e
